# Saved Collections PRD

Version: 1.0

---

# 1. Purpose

Saved Collections allow clients to organize inspiration, compare professionals, and return to ideas before making a booking decision.

Collections help bridge the gap between discovery and booking.

---

# 2. Problem Statement

Clients often save inspiration across multiple places:

- Instagram Saves
- Pinterest boards
- Screenshots
- Camera Roll
- Notes apps

This creates fragmented planning and makes it difficult to connect inspiration with the professionals who can actually deliver it.

Atlas keeps inspiration and booking in the same ecosystem.

---

# 3. Goals

Saved Collections should:

- Help users organize inspiration.
- Encourage return visits.
- Increase profile revisits.
- Increase booking conversion.
- Make planning easier.

---

# 4. Non-Goals

Saved Collections are not intended to:

- Replace Pinterest.
- Become cloud storage.
- Replace galleries or albums.

Their purpose is to support beauty decision-making.

---

# 5. User Stories

## Client

As a client I want to:

- Save posts I like.
- Save professional profiles.
- Create multiple collections.
- Rename collections.
- Delete collections.
- Share collections.
- Book directly from a saved item.

## Professional

As a professional I want to:

- Know when my work is frequently saved (aggregated insights).
- Create content worth saving.

---

# 6. Functional Requirements

## Collections

Clients can:

- Create unlimited collections.
- Rename collections.
- Delete collections.
- Reorder collections.

Examples:

- Summer Nails
- Wedding
- Vacation
- Autumn Ideas
- Christmas

---

## Saved Items

Users can save:

- Feed posts
- Professional profiles
- Styles
- Services (future)

---

## Collection Management

Users can:

- Add items.
- Remove items.
- Move items between collections.
- Sort items.

---

## Sharing

Users can:

- Share a collection via link.
- Send collections to friends.
- Use collections during consultations with professionals.

---

## Privacy

Collections support:

- Private (default)
- Shared via link

Public collections may be introduced in the future.

---

# 7. User Flow

Client discovers inspiration

↓

Saves post

↓

Adds it to a collection

↓

Returns later

↓

Compares professionals

↓

Books appointment

---

# 8. Success Metrics

Primary

- Save rate
- Collection creation rate
- Return rate
- Saved → Booking conversion

Secondary

- Collection sharing
- Average items per collection
- Repeat visits

---

# 9. Edge Cases

- Deleted posts
- Deleted professionals
- Duplicate saves
- Empty collections
- Broken shared links
