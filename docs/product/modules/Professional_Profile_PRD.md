# Professional Profile PRD

Version: 1.0

---

# 1. Purpose

The Professional Profile is the primary decision-making page within Atlas.

Its purpose is to help clients evaluate, trust, and book a beauty professional while giving professionals a complete digital presence to showcase their business.

The profile should answer every important question a client has before booking.

---

# 2. Problem Statement

Today, clients often need to visit multiple platforms to evaluate a professional:

- Instagram for portfolio
- Google Maps for reviews
- Direct Messages for questions
- Separate booking pages
- Personal websites
- WhatsApp for availability

This fragmented experience creates friction and reduces trust.

Atlas brings these elements together in one place.

---

# 3. Goals

The profile should help clients:

- Build trust.
- Evaluate quality.
- Understand services.
- View previous work.
- Read reviews.
- Check availability.
- Book confidently.

For professionals, the profile should:

- Showcase expertise.
- Increase visibility.
- Convert profile visitors into bookings.
- Build a long-term reputation.

---

# 4. Non-Goals

The profile is not intended to:

- Replace the Feed.
- Replace Search.
- Replace messaging.

Its primary role is evaluation and conversion.

---

# 5. User Stories

## Client

As a client I want to:

- View a professional's portfolio.
- Read reviews.
- See available services.
- Understand pricing.
- Check availability.
- Save the profile.
- Share the profile.
- Contact the professional if needed.
- Book directly.

## Professional

As a professional I want to:

- Present my work professionally.
- Highlight my experience.
- Display my services and prices.
- Show my availability.
- Receive bookings.
- Build trust through reviews.

---

# 6. Functional Requirements

The profile includes:

## Identity

- Profile photo
- Business name
- Display name
- Biography
- Verification status
- Languages spoken

---

## Portfolio

- Photos
- Videos (future)
- Collections
- Before & after examples
- Tagged styles

---

## Services

- Service list
- Duration
- Starting prices
- Add-ons

---

## Availability

- Working hours
- Calendar preview
- Next available appointment

---

## Reputation

- Overall rating
- Review count
- Review breakdown
- Client photos
- Repeat client indicator

---

## Trust Signals

- Verified professional
- Years of experience
- Certifications
- Response time
- Booking completion rate
- Cancellation rate

---

## Actions

Clients can:

- Book
- Follow
- Save
- Share
- Contact
- Report profile
