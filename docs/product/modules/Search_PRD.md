# Search PRD

Version: 1.0

---

# 1. Purpose

Search enables clients to quickly find the right beauty professional based on their intent.

Unlike the Feed, which supports exploration and inspiration, Search supports focused decision-making.

The objective is to minimize the time between a user's search and a successful booking.

---

# 2. Problem Statement

Today, clients search across multiple disconnected platforms:

- Instagram
- Google Maps
- TikTok
- Facebook groups
- Recommendations from friends
- Messaging apps

Each platform provides only part of the information required to make a confident booking decision.

Atlas unifies discovery, trust, and booking into a single search experience.

---

# 3. Goals

Search should allow users to:

- Find nearby professionals.
- Search by beauty category.
- Search by style.
- Search by service.
- Search by city or neighborhood.
- Filter by price.
- Filter by rating.
- Filter by availability.
- Compare professionals.
- Start booking immediately.

---

# 4. Non-Goals

Search is not intended to:

- Replace the Feed.
- Replace recommendations.
- Replace messaging.

Its responsibility is helping users actively find professionals.

---

# 5. User Stories

## Client

As a client I want to:

- Find professionals near me.
- Search by style.
- Search by city.
- Filter by distance.
- Filter by price.
- Filter by rating.
- Filter by availability.
- View portfolios before booking.
- Save professionals.
- Book directly from search results.

## Professional

As a professional I want to:

- Appear in relevant searches.
- Be discovered by nearby clients.
- Rank higher because of quality rather than popularity.
- Receive more qualified bookings.

---

# 6. Functional Requirements

Search supports:

- Keyword search
- Category search
- Style search
- Location search
- Distance radius
- City selection
- Availability filters
- Price filters
- Rating filters
- Service filters
- Sorting options
- Saved searches
- Recent searches
- Professional cards with key information

---

# 7. User Flow

Client opens Search

↓

Client enters search or selects filters

↓

Ranking System retrieves eligible professionals

↓

Results displayed

↓

Client opens professional profile

↓

Portfolio reviewed

↓

Booking initiated

---

# 8. Success Metrics

Primary Metrics

- Search success rate
- Search → Profile conversion
- Search → Booking conversion

Secondary Metrics

- Average search time
- Filter usage
- Saved professionals
- Repeat searches
- Booking completion rate

---

# 9. Edge Cases

- No professionals found
- Very small markets
- High-demand periods
- Incomplete professional profiles
- Fake locations
- Outdated availability
- Misspelled search queries
- Professionals serving multiple cities
